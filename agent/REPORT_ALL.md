# **图书推荐智能代理项目报告**

## **1. 项目概述**

本项目是一个基于 `LangGraph` 构建的智能图书推荐代理。它能够理解用户的自然语言指令，进行图书搜索，并根据用户的历史记录和偏好提供个性化的图书推荐。

## **2. 核心功能**

*   **智能图书搜索**:
    *   支持按书名、作者、类型等多种维度进行搜索。
*   **多维度图书推荐**:
    *   **基于作者推荐**: 推荐同一作者的其他作品。
    *   **基于类型推荐**: 推荐同一类型的其他图书。
    *   **基于知识图谱的智能推荐**: 利用内置的知识图谱（作者、类型、风格等关系）进行更深层次的推荐。
*   **个性化服务**:
    *   **用户偏好管理**: 记录和更新用户的喜好（喜欢的作者、类型等）。
    *   **阅读趋势分析**: 分析用户的阅读历史，构建用户画像。
*   **可解释性**:
    *   为每一本推荐的图书提供详细的推荐理由。

## **3. 技术实现**

*   **核心框架**: 使用 `LangGraph` 构建代理的工作流程（workflow），管理状态和逻辑流转。
*   **大语言模型 (LLM)**: 集成 `DeepSeek` 模型作为推理核心，理解用户意图并调用工具。
*   **状态管理**: 通过 Pydantic 模型 (`BookRecommendationState`) 定义了严谨的状态，用于在整个对话过程中跟踪用户信息、搜索历史、推荐结果等。
*   **工具集**: 定义了一系列功能明确的工具（`book_tools.py`），供大语言模型调用，以完成具体任务（如 `search_books`, `recommend_by_author` 等）。

## **4. 项目结构**

```
/
├── book_agent.py           # 代理核心逻辑，定义了工作流和节点
├── book_state.py           # 定义了代理的状态（Pydantic模型）
├── book_tools.py           # 封装了所有与图书相关的工具函数
├── book_run.py             # 项目启动脚本（推荐）
├── start_agent.py          # 另一个项目启动脚本
├── config.py               # 配置文件，用于设置API密钥和模型参数
├── requirements.txt        # Python依赖列表
└── BOOK_README.md          # 项目的详细说明文档
```

## **5. 如何运行**

1.  **安装依赖**:
    ```bash
    pip install -r requirements.txt
    ```
2.  **启动项目**:
    *   **交互模式**:
        ```bash
        python book_run.py
        ```
    *   **演示模式**:
        ```bash
        python book_run.py demo
        ```

## **6. 总结**

本项目是一个功能完整、结构清晰的智能代理应用案例。它有效地结合了 `LangGraph` 的工作流管理能力、大语言模型的自然语言理解能力以及自定义工具的执行能力，为用户提供了一个智能、个性化的图书推荐服务。代码的模块化程度高，易于理解和扩展。
